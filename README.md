# DrowsyGuard-Real-Time-Driver-Drowsiness-Detection-System
DrowsyGuard adalah sistem berbasis Computer Vision yang mendeteksi kantuk secara real-time menggunakan analisis mata (Eye Aspect Ratio). Dibangun dengan OpenCV dan dlib, sistem ini memberikan peringatan otomatis saat pengemudi terdeteksi mengantuk untuk meningkatkan keselamatan berkendara.
